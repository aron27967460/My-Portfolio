Style Guide

ELEMENT LIST
Navigation Button In-text Link Submit/Reset Button
Text Input Field Headings Paragraph Bulleted Lists Numbered Lists

NAVIGATION BUTTON

CSS selector:

.nav-button

Sample code:

<a href="#" class="nav-button">Default Button</a>

Rendered element:

Default

IN-TEXT LINK

CSS selector:

.intext-link

Sample code:

<a href="#" class="intext-link">In-text Link</a>

Rendered element:

In-text Link

SUBMIT/RESET BUTTON

CSS selector:

.d-button

Sample code:

<a class="d-button">DEFAULT BUTTON<a>

Rendered element:

DEFAULT BUTTON

TEXT INPUT FIELD

CSS selector:

.text-input

Sample code:

<input type="text" value="type here" class="text-input">

Rendered element:

HEADINGS

CSS selector:

#slogan, h1, h2, h3

Sample code:

<h1 id="slogan">Intro</h1>
<h1>Header</h1>
<h2>Subheader</h2>
<h3>Project Title</h3>

Rendered element:

Intro

Header

Subheader

Project Title

PARAGRAPH

CSS selector:

p

Sample code:

<p>Paragraph is here!</p>

Rendered element:

Lorem ipsum dolor sit amet, consectetur adipiscing elit. Etiam mattis nulla eu pharetra cursus. Nunc sed varius enim. Mauris vitae ullamcorper orci.

BULLETED LIST

CSS selector:

ul

Sample code:

<ul class="bulleted-lists"><item one><item two><item three></ul>

Rendered element:

  • item one
  • item two
  • item three

NUMBERED LIST

CSS selector:

ol

Sample code:

<ol class="ordered-lists"><item one><item two><item three></ol>

Rendered element:

  1. item one
  2. item two
  3. item three

MAIN NAVIGATION

CSS selector:

.nav-bar, .nav-button

Sample code:

<nav class="container nav-bar right"> <a href="index.html" class="logo"><img src="images/logo.png" alt="logo" style="width:80px;"></a> <a href="about-me.html" class="nav-button">About</a> <a href="index.html" class="nav-button">Work</a> <a href="files/Aron-Resume-4.0.pdf" target="_blank" class="nav-button last-nav-button">Resume</a> </nav>

Rendered element:

About Work Resume
